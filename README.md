# Estudo de Viabilidade Construtiva — Creare Arquitetura

Ferramenta de viabilidade construtiva com leitura automática de uso do solo via Gemini AI.

## Como usar

Acesse o link do GitHub Pages deste repositório.

## Configuração

1. Obtenha uma chave API gratuita em [aistudio.google.com](https://aistudio.google.com)
2. Cole a chave no campo indicado na ferramenta
3. Faça upload do documento de uso do solo (PDF ou foto)
4. O formulário será preenchido automaticamente
